Setup Instruction:

  1. Install "Send to Feishu" from Chrome Web Store.
  2. Create a Feishu App in the Feishu Open Platform.
  3. Get the App ID and App Secret from your Feishu App's configuration page.
  4. Choose the Receive ID Type and get the corresponding Receive ID (user or chat ID).
  5. Fill in the App ID, App Secret, Receive ID, and select the Receive ID Type in the extension settings page.
  6. If everything is working correctly, this page will display '"Send to Feishu" configured successfully!'.
